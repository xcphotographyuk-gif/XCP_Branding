## How to import a JSON file into Elementor

**On the page you are editing:**
1. Open the page → click **Edit with Elementor**
2. Click the **folder icon** (bottom-left of the Elementor panel) → **Import Template**
3. Select the `.json` file you downloaded from this repo
4. Click **Insert** — the section appears on your page
5. Repeat for each section in order

**To download a file from GitHub:**
- Click the link below → page opens → click the **Raw** button (top-right) → right-click → Save As → save as `.json`

## ✏️ After each import — things to check and fix

When a section loads, quickly check these before moving to the next:

- [ ] **Spelling** — scan every heading and paragraph
- [ ] **Your name** — confirm it says "Xavier" / "XCP Photography" not placeholder text
- [ ] **Phone / email** — replace any placeholder contact details with your real ones
- [ ] **Prices** — confirm £1,495 floor is showing correctly where relevant
- [ ] **Images** — placeholder images will show; add yours from the `images/` folder in this repo
- [ ] **Colours** — should be black `#000000`, warm off-white `#E8E4DC`, white `#FFFFFF`
- [ ] **Font** — Raleway only (check the Elementor global font setting if anything looks wrong)

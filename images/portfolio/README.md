## Rules:
- Minimum 12 portfolio images before going live
- Real work only — no stock images, no AI-generated images
- Mix of subjects: brand, music/nightlife, documentary, portrait, hospitality
- JPG, compress to under 150KB each (squoosh.app)

## How to add on desktop:
Drop images into this folder → upload to WordPress Media Library → swap placeholders in Elementor.

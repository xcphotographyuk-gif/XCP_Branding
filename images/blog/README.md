## Rules:
- 1200 × 628px is the ideal ratio for blog thumbnails and social sharing (1.91:1)
- Choose an image from your portfolio that relates to the post topic
- JPG, compress to under 200KB each
- These images also appear when the post is shared on LinkedIn/Facebook

## How to add on desktop:
Drop images here → upload to WordPress Media Library → set as Featured Image on each post.

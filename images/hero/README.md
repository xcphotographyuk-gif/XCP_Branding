# Images for XCP Photography — hero

Full-bleed background images for every page hero section.

## Files to add:

| Filename | Page | Dimensions | Notes |
|---|---|---|---|
| `homepage-hero.jpg` | Homepage (HomeP1) | 1920 × 1080px | Your single strongest image |
| `homepage-parallax.jpg` | Homepage (HomeP2) | 1920 × 1200px | Dark, moody, cinematic |
| `about-hero.jpg` | About page | 1920 × 1080px | Xavier on a shoot |
| `services-hero.jpg` | Services page | 1920 × 1080px | Brand shoot environment |
| `portfolio-hero.jpg` | Portfolio page | 1920 × 1080px | Strong editorial |
| `contact-hero.jpg` | Contact page | 1920 × 1080px | Atmospheric |
| `blog-hero.jpg` | Stories/Blog page | 1920 × 1080px | Clean, content-forward |

## Rules:
- JPG format, compress to under 300KB each (use squoosh.app — free, in-browser)
- Dark tones preferred — text overlays are white and need contrast
- Landscape orientation only for these slots

## How to add on desktop:
Drop images into this folder → upload to WordPress Media Library → assign in Elementor.

# Images for XCP Photography — stationery

Place your logo files here. These are used in Adobe Express templates,
the WordPress footer, email signature, and all printed materials.

## Files to add:

| Filename | What it is | Size |
|---|---|---|
| `XCP_Logo_White.png` | White logo, transparent background | 600 × 160px minimum |
| `XCP_Logo_Dark.png` | Dark/black logo, transparent background | 600 × 160px minimum |
| `XCP_Logo_White.svg` | Vector white logo (scales to any size) | — |
| `XCP_Logo_Dark.svg` | Vector dark logo (scales to any size) | — |

## How to add on desktop:
Save your logo files to this folder, then upload to WordPress via Media → Add New.
Use XCP_Logo_White.png anywhere the background is black (#000000).
Use XCP_Logo_Dark.png anywhere the background is white or light.

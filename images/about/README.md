# Images for XCP Photography — about

Images for the About page sections — hero and process steps.

## Files to add:

| Filename | Slot | Dimensions | What to use |
|---|---|---|---|
| `about-hero.jpg` | About page hero background | 1920 × 1080px | Xavier working — behind-the-scenes or environmental portrait |
| `process-01-discovery.jpg` | Process step 1 — Discovery | 800 × 600px | Conversation / consultation moment |
| `process-02-planning.jpg` | Process step 2 — Planning | 800 × 600px | Shot list / preparation |
| `process-03-shoot.jpg` | Process step 3 — The Shoot | 800 × 600px | On-shoot action — Xavier with subject |
| `process-04-delivery.jpg` | Process step 4 — Delivery | 800 × 600px | Gallery on screen / delivered result |

## Rules:
- JPG, compress to under 150KB for process steps, under 300KB for hero
- Process images should feel documentary — real moments, not posed setups
- These are about showing how you work, not just what you shoot

## How to add on desktop:
Drop images here → upload to WordPress Media Library → swap in Elementor (About page editor).

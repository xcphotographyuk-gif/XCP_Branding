# Images for XCP Photography — services

Images for the Services page — hero and individual package cards.

## Files to add:

| Filename | Slot | Dimensions | What to use |
|---|---|---|---|
| `services-hero.jpg` | Services page hero | 1920 × 1080px | Strong brand shoot — editorial, confident |
| `package-foundation.jpg` | Brand Foundation package card | 800 × 600px | Headshot / portrait work |
| `package-elevation.jpg` | Brand Elevation package card | 800 × 600px | Team or brand environment shoot |
| `package-transformation.jpg` | Brand Transformation package card | 800 × 600px | Full brand campaign — editorial range |

## Rules:
- Each package image should visually represent the scale of that package
- Foundation = focused, intimate | Elevation = wider, more variety | Transformation = full scale
- JPG, compress to under 200KB for package cards, under 300KB for hero

## How to add on desktop:
Drop images here → upload to WordPress Media Library → swap placeholders in Elementor (Services page).

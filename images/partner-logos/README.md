## Rules:
- White logo, transparent background — PNG only
- 320 × 120px minimum (640 × 240 at 2× for retina screens)
- If you can't find a white version: open the logo in Canva or Photoshop → convert to white → export as PNG with transparency
- Only use logos where you have the client's permission (implied by previous work together)

## Hotel du Vin note:
The Hotel du Vin credential block in the About page uses gold `#d4af37` — this is the ONLY place on the site where gold is used. All other credentials use platinum `#E8E4DC`.

## How to add on desktop:
Drop logo files here → upload to WordPress Media Library → assign in Elementor (HomeP5 and About P1 partner/credential sections).

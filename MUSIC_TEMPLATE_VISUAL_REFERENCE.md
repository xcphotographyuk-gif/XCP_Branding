## Section-by-Section Breakdown

### 1. Hero Section
- **Height:** 60% of viewport (60vh)
- **Background:** Your concert photo (dark overlay for text readability)
- **Content:**
  - Category label: "MUSIC" (gold color)
  - Post title (large, white, bold)
  - Date and location (white)
- **Position:** Text at bottom-left with shadow for visibility

### 2. Article Content
- **Max Width:** 900px (centered)
- **Background:** White
- **Padding:** 80px top/bottom
- **Contains:**
  - Intro paragraph (20px font, italic, larger)
  - Main body (18px font, 1.8 line height)
  - Byline section with border-top

### 3. Gallery Heading
- **Max Width:** 1200px (centered)
- **Background:** Light grey (#F5F5F5)
- **Content:**
  - Gallery title (36px heading)
  - Event subtitle (16px)
  - Copyright notice (14px)

### 4. Photo Gallery
- **Layout:** CSS Grid
- **Columns:** 3 (desktop) → 2 (tablet) → 1 (mobile)
- **Gap:** 15px between images
- **Aspect Ratio:** 3:2 (landscape)
- **Lightbox:** Click to view full-size
- **Images:** 12 placeholders (add/remove as needed)

### 5. Tour Schedule
- **Max Width:** 900px (centered)
- **Background:** White
- **Format:** Date + Venue + City
- **Dividers:** Between each date
- **Font:** 16px, clean spacing

### 6. Artist Information
- **Max Width:** 900px (centered)
- **Background:** Light grey (#F9F9F9)
- **Contains:**
  - Heading (28px)
  - Biography text
  - Links to official sites

### 7. Supporting Act (Optional)
- **Max Width:** 900px (centered)
- **Background:** White
- **Contains:**
  - Act name heading
  - Info about performance
  - Links to their content
- **Note:** Can be deleted if not needed

## Color Scheme

```
Primary Background: #FFFFFF (white)
Secondary Background: #F5F5F5, #F9F9F9 (light grey)
Text Primary: #1a1a1a (near black)
Text Secondary: #333333 (dark grey)
Text Tertiary: #666666, #999999 (grey)
Accent: #FFD700 (gold - for "MUSIC" label)
Links: #0066cc (blue)
Borders: #eeeeee (light grey)
```

---

## Typography

```
Headings:
- H1 (Title): 48px (desktop) / 36px (tablet) / 28px (mobile)
- H2 (Gallery): 36px (desktop) / 28px (mobile)
- H3 (Sections): 32px (desktop) / 24px (mobile) / 28px/22px

Body Text:
- Intro: 20px, italic, 1.6 line-height
- Main: 18px, regular, 1.8 line-height
- Metadata: 16px, 14px for small text

Font Weights:
- Headings: 700 (bold)
- Body: 400 (regular)
- Category: uppercase, 2px letter-spacing
```

---

## Spacing

```
Section Padding:
- Large: 80px top/bottom
- Medium: 60px top/bottom
- Small: 40px top/bottom
- Mobile: 20px left/right

Content Spacing:
- Between paragraphs: 15-30px
- Between sections: Natural section breaks
- Gallery gap: 15px
```

---

## Responsive Breakpoints

```
Desktop (1200px+):
- 3-column gallery
- Full content widths
- Large typography

Tablet (768px - 1199px):
- 2-column gallery
- Adjusted content widths
- Medium typography

Mobile (< 768px):
- 1-column gallery
- Full-width stacked layout
- Smaller typography
- Reduced padding
```

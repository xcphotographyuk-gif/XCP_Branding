# Music Post Template - Complete Guide

## Overview
The XCP Music Post Template is designed for publishing concert reviews, live music photography, and music event coverage on your WordPress website. It provides a professional, consistent layout for all music-related blog posts.

---

## Template File
**File:** `XCP_Music_Post_Template.json`

---

## What's Included

### Template Sections

1. **Hero Image Section** (60vh height)
   - Full-width featured concert image with dark overlay
   - Category label ("MUSIC")
   - Post title (Artist: Event at Venue)
   - Date and location metadata
   - Dramatic presentation with text shadow

2. **Article Content Section** (900px max width)
   - Intro paragraph (italicized, larger font for emphasis)
   - Main article body (formatted for easy reading)
   - Author byline and credits (Words & Images)
   - Clean, professional typography

3. **Gallery Heading Section**
   - Gallery title with artist name
   - Event and venue information
   - Copyright notice

4. **Photo Gallery Section** (1200px max width)
   - Responsive grid layout (3 columns → 2 → 1)
   - 12 image placeholders (add or remove as needed)
   - Lightbox functionality for full-size viewing
   - 3:2 aspect ratio for consistency
   - 15px gap between images

5. **Tour Schedule Section** (900px max width)
   - Formatted list of tour dates
   - Venue and location details
   - Clean, scannable layout with dividers

6. **Artist Information Section**
   - Artist biography and background
   - Notable achievements
   - Links to official websites/social media

7. **Supporting Act Section** (Optional)
   - Information about opening/supporting acts
   - Links to their music and tour dates
   - Can be removed if not needed

## How to Use This Template

### Step 1: Create New Post
1. Go to **WordPress Dashboard → Posts → Add New**
2. Enter your post title (e.g., "Alabama 3: The Divine Intervention Tour at The Concorde 2")
3. Set **Category** to "Music" (create category if it doesn't exist)
4. Click **Edit with Elementor**

### Step 2: Import Template
1. In Elementor, click the **folder icon** (Template Library)
2. Go to **My Templates** tab
3. Find **XCP Music Post Template**
4. Click **Insert**

### Step 3: Customize Content

#### Hero Section:
- **Background Image:** Click section → Style → Background → Choose your best wide concert shot (1920x1080px recommended)
- **Title:** Edit to "Artist Name: Event Title at Venue Name"
- **Date/Location:** Update to actual show date and venue

#### Article Content:
- **Intro Paragraph:** Replace with your opening/lead paragraph (keep italicized style)
- **Main Body:** Paste your article content, maintaining paragraph breaks
- **Byline:** Update "Words: Author Name" and "Images: Xavier Clarke"

#### Gallery Section:
- **Gallery Title:** Update to artist name
- **Gallery Subtitle:** Update venue and event details
- **Images:** Click gallery widget → Click each placeholder → Choose your concert photos
  - **Tip:** Upload 10-15 of your best shots
  - **Recommended size:** 1200x800px or larger
  - **Format:** JPG (compressed for web)

#### Tour Schedule:
- **Dates:** Edit the list to match actual tour dates
- **Add/Remove:** Copy/paste the paragraph format for more dates
- **Remove Section:** If no tour info available, delete this entire section

#### Artist Information:
- **Bio:** Add or edit artist background information
- **Links:** Update placeholder links to actual websites

#### Supporting Act:
- **Keep/Remove:** Delete this section if there was no supporting act
- **Customize:** Update with opening act details if applicable

### Step 4: SEO & Settings
1. **Featured Image:** Set the same hero image as post featured image
2. **Excerpt:** Write a brief summary (150-200 characters)
3. **Categories:** Ensure "Music" is selected
4. **Tags:** Add relevant tags (artist name, venue, genre, etc.)

### Step 5: Publish
1. **Preview** the post to check layout on desktop and mobile
2. **Update** any final tweaks
3. **Publish** when ready

## Content Guidelines

### Writing Style
- **Opening:** Hook readers with atmosphere and excitement
- **Body:** Balance detail with readability
- **Include:**
  - Performance highlights
  - Venue atmosphere
  - Audience energy
  - Notable moments
  - Overall recommendation

### Photo Selection
- **Hero Image:** Most dramatic, wide shot (crowd, stage, or performer)
- **Gallery:**
  - Mix of close-ups and wide shots
  - Artist performance shots
  - Crowd/atmosphere shots
  - Lighting and stage design
  - Supporting act (if applicable)

### Optimization
- **Images:** Compress before upload (use TinyPNG or similar)
- **File names:** Use descriptive names (artist-venue-date-01.jpg)
- **Alt text:** Add for every image (good for SEO and accessibility)

## Template Customization Options

### Easy Modifications

**Change Hero Height:**
- Click hero section → Layout → Min Height → Adjust value

**Change Content Width:**
- Click section → Layout → Content Width → Adjust value

**Change Gallery Columns:**
- Click gallery widget → Layout → Columns → Choose 2, 3, or 4

**Change Colors:**
- Category label: Edit text color in HTML
- Headings: Click heading → Style → Text Color
- Background: Click section → Style → Background Color

**Add/Remove Sections:**
- Delete unwanted sections (right-click section → Delete)
- Duplicate sections (right-click → Duplicate)

## Mobile Responsiveness

The template is fully responsive with automatic adjustments:

- **Desktop (1200px+):** 3-column gallery, full-width hero
- **Tablet (768-1199px):** 2-column gallery, adjusted text sizes
- **Mobile (< 768px):** Single-column layout, stacked content

**To Test:**
1. Click the responsive mode icon in Elementor
2. Switch between Desktop/Tablet/Mobile views
3. Adjust any element-specific mobile settings if needed

## Tips for Success

### Before Publishing:
- [ ] Spell-check all content
- [ ] Verify all tour dates and links
- [ ] Test lightbox gallery functionality
- [ ] Check mobile layout
- [ ] Add meta description for SEO
- [ ] Set featured image
- [ ] Add relevant categories and tags

### Content Strategy:
- **Timing:** Publish within 24-48 hours of show (while it's fresh)
- **Promotion:** Share on social media with gallery link
- **Cross-promotion:** Link to portfolio, other concert posts
- **SEO:** Use location + artist name in title
- **Engagement:** Encourage comments, tag artists/venues

### Photography Tips:
- **Bring both systems:** M43 for compact mobility, FF for low light
- **Variety:** Mix wide, medium, and close-up shots
- **Edit consistently:** Maintain visual style across gallery
- **Copyright:** Always watermark or embed copyright in metadata

## Troubleshooting

### Images Not Showing:
- Verify images are uploaded to Media Library
- Check file paths in gallery widget
- Clear Elementor cache: Tools → Regenerate CSS

### Layout Issues:
- Check responsive settings for each element
- Verify section width settings
- Clear browser cache

### Gallery Not Opening in Lightbox:
- Gallery widget → Settings → Link To: "Media File"
- Open Lightbox: "Yes"
- Update and republish

### Form Issues (if adding contact):
- Use Elementor Form widget
- Configure email action
- Test submission

## Updating the Template

### Make Changes:
1. Edit any post using this template
2. Make your improvements
3. Select all sections (Ctrl/Cmd + A)
4. Right-click → Save as Template
5. Name it "XCP Music Post Template v2"
6. Use updated version for new posts

### Version Control:
- Keep original template as backup
- Name new versions clearly
- Document changes in template description

## USING THESE TEMPLATES IN ADOBE EXPRESS:

### Step-by-Step:

1. **Choose template size** (provided above for each type)

2. **Set up brand kit in Adobe Express:**
   - Add colors: #2DD4BF, #1F2937, #F9FAFB, #F59E0B
   - Upload logo
   - Set fonts: Inter/Montserrat

3. **For each template:**
   - Start with blank canvas at correct size
   - Add image placeholder (shape/box with note "Add your photo here")
   - Add text boxes with provided copy
   - Style with brand colors/fonts
   - Leave flexibility for YOUR photos

4. **Save as template** in Adobe Express for reuse

5. **Create variations:**
   - Swap colors (teal vs charcoal backgrounds)
   - Change layouts (text left vs right)
   - Try different photos

## BATCH CREATION WORKFLOW:

**Week 1: Set up templates**
- Create 3-4 templates in Adobe Express
- Save them for reuse

**Week 2: Create content library**
- Use templates to create 20-30 graphics

**Week 3: Populate with photos**
- After a shoot, add photos to templates

- Schedule posts

- Refresh messaging seasonally
- Test and optimize based on engagement

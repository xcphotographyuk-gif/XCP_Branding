# XCP_Branding
XC-Photography Website - Elementor Homepage Template

## Overview
This repository contains comprehensive Elementor homepage layout templates for XC Photography's branding and documentary photography website with a narrative-driven, cinematic design approach.

## Files Included

### Main Templates
- **elementor-homepage-template.json** - Complete Elementor template with all sections ready for import
- **preview.html** - Interactive HTML preview of the complete template design
- **TEMPLATE_DOCUMENTATION.md** - Comprehensive installation and customization guide

### Individual Section Templates
- **XCP_HomeP1.json** - Top Hero Section with striking visuals and primary CTA
- **XCP_HomeP2_TextBoxes.json** - Three-column services/features section
- **XCP_HomeP4.json** - Bottom Hero Section with final call-to-action
- **XCP_HomeP5_Footer.json** - Complete footer with navigation, social links, and copyright

## Quick Start

1. Import `elementor-homepage-template.json` into your WordPress/Elementor installation
2. Follow the detailed instructions in `TEMPLATE_DOCUMENTATION.md`
3. Preview the design by opening `preview.html` in a web browser
4. Customize individual sections using the separate JSON templates if needed

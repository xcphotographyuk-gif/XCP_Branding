## Quick Start

### For Page Templates:
1. Review `COMPLETE_SITE_TEMPLATES.md` for all 12 page templates
2. Import page templates into WordPress/Elementor: **Templates → Import Template**
3. Create pages using imported templates
4. Replace placeholder images with your photography

### For Music Blog Posts:
1. Import `XCP_Music_Post_Template.json` to Elementor
2. Create new Post → Set Category to "Music"
3. Edit with Elementor → Insert Music Post Template
4. Use `Alabama3_Article_Content.md` as example content
5. Upload concert photos to gallery section
6. Publish!

**See `MUSIC_POST_TEMPLATE_GUIDE.md` for complete music post instructions.**

## Music Post Template - Quick Guide

The Music Post Template is designed for publishing concert photography and music event coverage on your blog.

### What's Included:
1. **Hero Image Section** - Full-width concert photo with title overlay
2. **Article Content** - Professional blog layout with intro and main text
3. **Photo Gallery** - 12-image responsive grid with lightbox
4. **Tour Schedule** - Formatted tour dates section
5. **Artist Info** - Biography and links
6. **Supporting Act** - Optional opening act section

### How to Use:
1. **Import:** Upload `XCP_Music_Post_Template.json` to Elementor Templates
2. **Create Post:** WordPress → Posts → Add New → Set Category: "Music"
3. **Edit:** Click "Edit with Elementor" → Insert Music Post Template
4. **Customize:**
   - Replace hero background with your best concert shot
   - Update title, date, and venue
   - Paste article content
   - Upload 10-15 gallery photos
   - Add tour dates and artist info
5. **Publish!**

### Sample Content:
Use the Alabama 3 concert article as a reference:
- `Alabama3_Article_Content.md` - Complete article with all sections
- `Alabama3_Alamy_Metadata.md` - Tags and captions for image uploads

**Full instructions:** See `MUSIC_POST_TEMPLATE_GUIDE.md`

# XC Photography Elementor Homepage Template

This template provides a professional, narrative-driven homepage layout for XC Photography's branding and documentary photography website with cinematic styling.

## Template Overview

### Sections Included

1. **Top Hero Section** (XCP_HomeP1.json)
   - Full-width cinematic background imagery
   - Central overlay with:
     - Bold H1 headline: "Branding & Documentary Photography That Demands Attention"
     - Descriptive subheading: "Iconic Imagery for Iconic Brands with an Edge"
   - Single prominent Call-to-Action button:
     - Primary CTA: "Book Your Free Consultation" (gold accent color with glow effect)
   - Optimized responsive typography for all devices
   - Dramatic gradient overlay for text readability

2. **Services/Features Section** (XCP_HomeP2_TextBoxes.json)
   - Three-column grid layout displaying core services
   - Each feature box includes:
     - Icon representing the service (camera, film, palette)
     - Bold heading
     - Descriptive paragraph
   - Services highlighted:
     - **Branding Photography** - Visual narratives for brand identity
     - **Documentary Storytelling** - Authentic moments and emotions
     - **Creative Direction** - Expert visual concepts
   - Hover effects with elevation animation
   - Gold accent border on each box
   - Fully responsive grid (stacks on mobile)

3. **Masonry Gallery Section**
   - Professional grid layout with three columns
   - Placeholder gallery entries:
     - "Striking Stories"
     - "Embracing Individuality"
     - "Timeless Beauty"
   - Hover animations (grow effect) for enhanced user experience
   - Responsive design for mobile and tablet views
   - Light background for visual separation

4. **Bottom Hero Section / Final CTA** (XCP_HomeP4.json)
   - Emotionally compelling call-to-action section
   - 80vh height with cinematic background
   - Components:
     - H2 headline: "Ready to Tell Your Story?"
     - Supportive descriptive paragraph
     - Prominent "Contact Us Now" button
   - Dark gradient overlay for text contrast
   - Perfect closing section to drive conversions

5. **Footer Section** (XCP_HomeP5_Footer.json)
   - Comprehensive three-column footer layout:
     - **Column 1**: Brand name, tagline, and description
     - **Column 2**: Quick navigation links (Home, About, Services, Portfolio, Contact)
     - **Column 3**: Social media icons (Facebook, Instagram, Twitter, LinkedIn, Pinterest)
   - Dark background (#1a1a1a) with gold accent top border
   - Separate copyright section with:
     - Copyright notice
     - Privacy Policy and Terms of Service links
   - Fully responsive (stacks to single column on mobile)
   - Hover effects on links and social icons

## Installation Instructions

### Prerequisites
- WordPress installation
- Elementor (Free or Pro version)
- Active theme compatible with Elementor

### Import Process

1. **Access Elementor Template Library**
   - Log in to your WordPress admin panel
   - Navigate to Templates > Saved Templates
   - Click "Import Templates"

2. **Upload Template**
   - **Option A: Import Complete Template**
     - Click "Choose File"
     - Select `elementor-homepage-template.json`
     - Click "Import Now"
   - **Option B: Import Individual Sections**
     - Import `XCP_HomeP1.json` for Top Hero
     - Import `XCP_HomeP2_TextBoxes.json` for Services/Features
     - Import `XCP_HomeP4.json` for Bottom Hero/CTA
     - Import `XCP_HomeP5_Footer.json` for Footer
     - Mix and match sections as needed

3. **Apply to Homepage**
   - Go to Pages > Add New (or edit existing homepage)
   - Click "Edit with Elementor"
   - Click the folder icon to open Template Library
   - Go to "My Templates" tab
   - Find "XC Photography Homepage - Complete"
   - Click "Insert"

## Customization Guide

### Replace Placeholder Images

1. **Top Hero Section Background**
   - Click on the Top Hero section
   - Go to Style tab
   - Under Background > Image, upload your cinematic photography
   - Recommended size: 1920x1080px or larger
   - Recommended format: JPG (optimized for web)
   - Use high-contrast images for text readability

2. **Bottom Hero Section Background**
   - Click on the Bottom Hero section
   - Go to Style tab
   - Upload a different compelling image
   - Recommended size: 1920x900px or larger
   - Choose emotionally resonant imagery

3. **Gallery Images**
   - Click on each image widget
   - Replace placeholder URLs with your actual portfolio images
   - Recommended sizes:
     - Varying heights (600-900px) for masonry effect
     - Width: 600px minimum
   - Formats: JPG or WebP for optimal performance

4. **Service Icons**
   - Click on each icon widget in the Text Boxes section
   - Choose from Font Awesome icon library
   - Or upload custom SVG icons
   - Keep icons simple and recognizable

### Customize Colors

**Brand Colors Used:**
- Gold Accent: `#d4af37` (Primary CTA buttons, icons, borders)
- Dark Background: `#1a1a1a` (Footer, text overlays)
- Light Gray: `#fafafa` (Service boxes, gallery backgrounds)
- White: `#ffffff` (Primary text and elements)
- Medium Gray: `#666666` (Body text)
- Very Dark: `#0f0f0f` (Copyright section)

**To Change Colors:**
1. Click on any element
2. Go to Style tab
3. Update color values in the color picker

### Update Text Content

1. **Headlines**
   - Click on heading widget
   - Edit text directly in the editor
   - Maintain bold, attention-grabbing language
   - Keep headlines concise for mobile readability

2. **Service Descriptions**
   - Click on text editor widgets in the Text Boxes section
   - Update with your actual service descriptions
   - Keep paragraphs focused and benefit-driven
   - Maintain consistent tone across all three boxes

3. **CTAs**
   - Click on button widgets
   - Update "Link" field with actual URLs (e.g., contact form, booking page)
   - Customize button text as needed
   - Test all links after customization

4. **Footer Content**
   - Update brand description in Column 1
   - Modify navigation links in Column 2 to match your site structure
   - Update social media URLs in Column 3
   - Customize copyright year and legal links

### Responsive Design

The template is fully mobile-responsive with breakpoints for:
- **Desktop** (1025px+): Full layout as designed with three-column grids
- **Tablet** (769px-1024px): Adjusted font sizes, maintained grid layouts
- **Mobile** (≤768px): Single column layout, stacked elements, optimized touch targets

**Responsive Features:**
- Hero sections maintain full viewport height on all devices
- Text boxes stack vertically on mobile for easy reading
- Footer columns stack into single column on mobile
- Font sizes scale appropriately across devices
- Touch-friendly button sizes on mobile (minimum 44px)
- Optimal line lengths for readability

**To Fine-tune Responsive Settings:**
1. Click responsive mode icons in Elementor editor (desktop/tablet/mobile)
2. Adjust settings for each device independently
3. Preview changes before publishing
4. Test on actual devices when possible

## Design Specifications

### Typography
- **Primary Font:** Montserrat (headings, buttons, brand elements)
- **Secondary Font:** Open Sans (body text, descriptions)
- **Heading Sizes:**
  - H1: 56px (desktop), 42px (tablet), 32px (mobile)
  - H2: 48px (desktop), 38px (tablet), 28px (mobile) - Bottom Hero
  - H2: 42px (desktop), 36px (tablet), 28px (mobile) - Gallery Section
  - H3: 26px (desktop), 24px (tablet), 22px (mobile) - Service Headings
  - H3: 20px (all devices) - Gallery Captions
  - H4: 18px (footer headings)
- **Body Text:** 16px (services), 14-15px (footer), 13px (copyright)
- **Line Heights:** 1.2-1.3 (headings), 1.6-1.8 (body text)

### Spacing
- **Section Padding:** 
  - Hero Sections: Full viewport height (100vh top, 80vh bottom)
  - Content Sections: 80px top/bottom, 20px left/right
  - Footer: 60px top, 40px bottom
  - Copyright: 25px top/bottom
- **Element Margins:** 
  - Between major elements: 20-45px
  - Icon to heading: 25px
  - Heading to text: 20px
  - Text to button: 40-45px
- **Grid Gaps:** 
  - Text boxes: 20px
  - Gallery items: Narrow (tight masonry)
  - Footer columns: 40px

### Effects
- **Text Shadows:** Applied to all hero headings for readability over images
- **Button Shadows:** Gold glow (rgba(212,175,55,0.4-0.5)) on primary CTAs
- **Hover Animations:**
  - Buttons: Grow effect with enhanced shadow
  - Text boxes: Translate up (-5px) with shadow
  - Gallery images: Scale (1.05)
  - Social icons: Translate up (-3px) with color change
- **Gradient Overlays:** Dark gradients on hero backgrounds (top to bottom)
- **Border Accents:** 3px gold bottom border on text boxes, 3px gold top border on footer

## Best Practices

### Image Optimization
- Compress all images before upload (TinyPNG, ShortPixel, ImageOptim)
- Use WebP format when possible for better compression
- Implement lazy loading for images below the fold
- Hero images: Optimize to 200-400KB for fast loading
- Gallery images: Keep individual images under 150KB
- Recommended total page size: Under 3MB for optimal performance

### SEO Considerations
- Update meta title: "Brand Name | Branding & Documentary Photography"
- Write compelling meta description (150-160 characters)
- Use descriptive alt text for all images (include keywords naturally)
- Maintain proper heading hierarchy (H1 → H2 → H3 → H4)
- Add schema markup for business information (LocalBusiness, Organization)
- Implement Open Graph tags for social media sharing
- Create XML sitemap and submit to search engines

### Accessibility
- Ensure sufficient color contrast (WCAG AA compliant)
- Add descriptive aria-labels to social media links
- Make sure all interactive elements are keyboard accessible
- Use semantic HTML throughout
- Test with screen readers
- Maintain minimum touch target size of 44x44px on mobile

### Performance
- Enable caching plugin (WP Rocket, W3 Total Cache, or similar)
- Use CDN for image and asset delivery (Cloudflare, StackPath)
- Minimize custom CSS/JS when possible
- Defer non-critical JavaScript
- Test with Google PageSpeed Insights (aim for 90+ score)
- Monitor Core Web Vitals (LCP, FID, CLS)
- Enable GZIP compression on server
- Optimize database regularly

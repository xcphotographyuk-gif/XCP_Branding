# XC Photography Elementor Homepage Template

This template provides a professional homepage layout for XC Photography's branding and documentary photography website.

## Template Overview

### Sections Included

1. **Hero Section**
   - Full-width cinematic background imagery
   - Central overlay with:
     - Bold H1 headline: "Branding & Documentary Photography That Demands Attention"
     - Descriptive subheading: "Iconic Imagery for Iconic Brands with an Edge"
   - Two prominent Call-to-Action buttons:
     - Primary CTA: "Book Your Free Consultation" (gold accent color)
     - Secondary CTA: "View Portfolio" (outlined white)

2. **Masonry Gallery Section**
   - Professional grid layout with three columns
   - Placeholder gallery entries:
     - "Striking Stories"
     - "Embracing Individuality"
     - "Timeless Beauty"
   - Hover animations for enhanced user experience
   - Responsive design for mobile and tablet views

3. **Placeholder Section**
   - Ready-to-customize content area
   - Suggested uses:
     - Client testimonials
     - Services overview
     - Team introduction
     - Brand story

## Installation Instructions

### Prerequisites
- WordPress installation
- Elementor (Free or Pro version)
- Active theme compatible with Elementor

### Import Process

1. **Access Elementor Template Library**
   - Log in to your WordPress admin panel
   - Navigate to Templates > Saved Templates
   - Click "Import Templates"

2. **Upload Template**
   - Click "Choose File"
   - Select `elementor-homepage-template.json`
   - Click "Import Now"

3. **Apply to Homepage**
   - Go to Pages > Add New (or edit existing homepage)
   - Click "Edit with Elementor"
   - Click the folder icon to open Template Library
   - Go to "My Templates" tab
   - Find "XC Photography Homepage"
   - Click "Insert"

## Customization Guide

### Replace Placeholder Images

1. **Hero Section Background**
   - Click on the Hero section
   - Go to Style tab
   - Under Background > Image, upload your cinematic photography
   - Recommended size: 1920x1080px or larger
   - Recommended format: JPG (optimized for web)

2. **Gallery Images**
   - Click on each image widget
   - Replace placeholder URLs with your actual portfolio images
   - Recommended sizes:
     - Varying heights (600-900px) for masonry effect
     - Width: 600px minimum
   - Formats: JPG or WebP for optimal performance

### Customize Colors

**Brand Colors Used:**
- Gold Accent: `#d4af37` (Primary CTA button)
- Dark Background: `#1a1a1a` (text overlays)
- Light Gray: `#fafafa` (gallery section background)
- White: `#ffffff` (text and secondary elements)

**To Change Colors:**
1. Click on any element
2. Go to Style tab
3. Update color values in the color picker

### Update Text Content

1. **Headlines**
   - Click on heading widget
   - Edit text directly in the editor
   - Maintain bold, attention-grabbing language

2. **CTAs**
   - Click on button widget
   - Update "Link" field with actual URLs
   - Customize button text as needed

### Responsive Design

The template is mobile-responsive with breakpoints for:
- Desktop: Full layout as designed
- Tablet: Adjusted font sizes and spacing
- Mobile: Single column layout, optimized touch targets

**To Fine-tune Responsive Settings:**
1. Click responsive mode icons in Elementor editor
2. Adjust settings for each device
3. Preview changes before publishing

## Design Specifications

### Typography
- **Primary Font:** Montserrat (headings, buttons)
- **Secondary Font:** Open Sans (body text, subheadings)
- **Heading Sizes:**
  - H1: 56px (desktop), 42px (tablet), 32px (mobile)
  - H2: 42px (desktop), 36px (tablet), 28px (mobile)
  - H3: 20px (consistent across devices)

### Spacing
- Section Padding: 80px top/bottom
- Element Margins: Consistent 15-50px based on hierarchy
- Grid Gap: Narrow (for tight masonry effect)

### Effects
- Text Shadow: Applied to hero text for readability
- Button Shadow: Gold glow on primary CTA
- Hover Animations: "Grow" effect on gallery images
- Gradient Overlay: Dark gradient on hero background

## Best Practices

### Image Optimization
- Compress all images before upload (TinyPNG, ShortPixel)
- Use WebP format when possible
- Lazy load images for better performance
- Recommended total page size: Under 3MB

### SEO Considerations
- Update meta title and description for homepage
- Use descriptive alt text for all images
- Maintain proper heading hierarchy (H1 → H2 → H3)
- Add schema markup for business information

### Performance
- Enable caching plugin (WP Rocket, W3 Total Cache)
- Use CDN for image delivery
- Minimize custom CSS/JS when possible
- Test with Google PageSpeed Insights

# Quick Start: Publishing Your Alabama 3 Concert Post

This guide will help you publish your first music blog post using the new Music Post Template in 5 minutes.

---

## Step 1: Import the Template (One-Time Setup)

1. **Log into WordPress**
2. **Go to:** Elementor ‚Üí Templates ‚Üí Saved Templates
3. **Click:** "Import Templates" button (top right)
4. **Choose File:** Select `XCP_Music_Post_Template.json` from this repository
5. **Click:** "Import Now"
6. **‚úì Done!** Template is now ready to use

---

## Step 2: Create Music Category (One-Time Setup)

1. **Go to:** Posts ‚Üí Categories
2. **Add New Category:**
   - Name: `Music`
   - Slug: `music`
3. **Click:** "Add New Category"
4. **‚úì Done!** Category is ready

---

## Step 3: Create Your Alabama 3 Post

### A. Create the Post
1. **Go to:** Posts ‚Üí Add New
2. **Title:** `Alabama 3: The Divine Intervention Tour at The Concorde 2`
3. **Category:** Check "Music"
4. **Click:** "Edit with Elementor" button

### B. Insert Template
1. **In Elementor:** Click the folder icon üìÅ (bottom left)
2. **Go to:** "My Templates" tab
3. **Find:** "XCP Music Post Template"
4. **Click:** "Insert"
5. **‚úì Template loads!** Now customize it...

---

## Step 4: Customize Content

### Hero Section (Top Image)
1. **Click on the hero section** (the big image at top)
2. **Left panel:** Style ‚Üí Background
3. **Upload your best wide concert shot** (1920x1080px recommended)
4. **Update Title:** Already correct if you used the suggested title
5. **Update Date/Location:**
   - Click the date text
   - Edit to: `March 30th, 2025 ‚Ä¢ Concorde 2, Brighton`

### Article Content
1. **Open:** `Alabama3_Article_Content.md` from this repository
2. **Scroll to:** "ARTICLE CONTENT" section
3. **Copy the intro paragraph** (starts with "It isn't often...")
4. **In Elementor:** Click the intro text area
5. **Paste:** Replace placeholder with your text
6. **Repeat for main article body** (copy from markdown file)
7. **Update byline:**
   - Words: Raphaela Clarke
   - Images: Xavier Clarke

### Photo Gallery
1. **Scroll down** to the gallery section in Elementor
2. **Click the gallery widget**
3. **Left panel:** Click the "+" icon to add images
4. **Upload 10-15 of your best concert photos**
5. **Arrange in order** (drag to reorder)
6. **Delete unused placeholders** (click X on each)

### Tour Schedule
1. **Scroll to tour dates section**
2. **Click the tour dates text**
3. **Copy tour dates** from `Alabama3_Article_Content.md`
4. **Paste:** Replace the placeholder dates

### Artist Information
1. **Scroll to "About the Artist" section**
2. **Click the bio text**
3. **Copy artist bio** from `Alabama3_Article_Content.md`
4. **Paste:** Replace placeholder text
5. **Update links** if you have official website URLs

### Supporting Act (The Dead Reds)
1. **Scroll to "Supporting Act" section**
2. **Update heading:** "The Dead Reds"
3. **Copy supporting act info** from `Alabama3_Article_Content.md`
4. **Paste:** Replace placeholder

## Troubleshooting

### Template Not Showing After Import?
- Refresh the Templates page
- Check it's in "My Templates" not "Blocks"
- Try logging out and back into WordPress

### Gallery Images Not Appearing?
- Make sure images are uploaded to Media Library first
- Click gallery widget ‚Üí Click each placeholder ‚Üí Choose image
- Update page after adding images

### Layout Looks Broken on Mobile?
- Click responsive icon in Elementor
- Switch to mobile view
- Adjust any specific mobile settings needed
- Most responsive settings are already configured

### Can't Find Category "Music"?
- Create it: Posts ‚Üí Categories ‚Üí Add New
- Make sure to check it when creating the post

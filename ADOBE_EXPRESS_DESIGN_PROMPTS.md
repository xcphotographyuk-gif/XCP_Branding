## Sales Brochures, Catalogues, and Marketing Materials

**Purpose:** Ready-to-use Adobe Express AI prompts for creating professional marketing materials. Each prompt includes image placeholder guidance and narrative suggestions.

**How to Use:**
1. Copy the prompt text
2. Paste into Adobe Express AI prompt field
3. Let AI generate the layout
4. Add your photos where marked `[BLANK]`
5. Copy/paste provided narrative text
6. Customize colors to match XCP brand (teal #2DD4BF, charcoal #1F2937)

## Design Tips for All Materials

### Brand Colors:
- Primary: Teal #2DD4BF
- Secondary: Charcoal #1F2937
- Accent: White #FFFFFF
- Optional: Light gray #F3F4F6 for backgrounds

### Fonts:
- Headlines: Montserrat Bold or similar sans-serif
- Body: Open Sans Regular or similar
- Keep it clean and modern

### Image Quality:
- Use only your best work
- High resolution (300 DPI for print)
- Consistent editing style
- Show variety but maintain cohesion

### White Space:
- Don't overcrowd
- Let images breathe
- Clean, modern aesthetic

### Print Specifications:
- PDF with 3mm bleed
- CMYK color mode
- 300 DPI resolution
- Embed all fonts

### Digital Specifications:
- PNG or JPG
- RGB color mode
- 150 DPI minimum
- Optimize file size for web

## Quick Start Workflow

**Creating Your First Brochure (30 minutes):**

1. **Prepare Images (10 min)**
   - Select 10 best portfolio images
   - Export high-res versions
   - Save in one folder

2. **Generate Design (5 min)**
   - Copy Tri-Fold Brochure prompt above
   - Paste into Adobe Express
   - Let AI generate layout

3. **Customize (10 min)**
   - Add your images where marked [BLANK]
   - Copy/paste narrative text provided
   - Adjust colors to XCP brand

4. **Export (5 min)**
   - PDF for print (300 DPI, CMYK)
   - PNG for digital sharing
   - Save both versions

**Done! Professional brochure ready to use.**

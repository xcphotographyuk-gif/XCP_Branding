## 🚀 HOW TO IMPORT INTO ELEMENTOR

### Method 1: Staged Import (Recommended - Avoids Large File Errors)

1. **In WordPress Dashboard:**
   - Go to Templates → Saved Templates → Import Templates

2. **Import Each Part Sequentially:**
   - Import `XCP_HomeP1_Hero_Gallery_FINAL.json` → Click "Import Now"
   - Import `XCP_HomeP2_Parallax_Text_FINAL.json` → Click "Import Now"
   - Import `XCP_HomeP3_Checkerboard_FINAL.json` → Click "Import Now"
   - Import `XCP_HomeP4_Stats_Process_FINAL.json` → Click "Import Now"
   - Import `XCP_HomeP5_Testimonials_CTA_FINAL.json` → Click "Import Now"
   - Import `XCP_ContactForm_Section.json` → Click "Import Now"

3. **Create Your Homepage:**
   - Pages → Add New → Edit with Elementor
   - Click folder icon → My Templates
   - Insert each section in order (drag from templates)

4. **Add Images:**
   - Click on each background image placeholder
   - Upload your actual photos
   - Recommended images:
     - Hero: Woman with whooshing hair, eye visible
     - Parallax: Triumph bike in snow
     - Gallery bars: Your best 4 portfolio pieces
     - Checkerboards: Brand/documentary work examples
     - Partner logos: Client/partner brand logos

5. **Publish:**
   - Review everything
   - Click "Publish"
   - Set as homepage in Settings → Reading → Homepage displays → Static page

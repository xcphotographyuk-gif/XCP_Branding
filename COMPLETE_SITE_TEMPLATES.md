## Import Instructions

### Step 1: Import All Templates to Elementor

1. Go to **WordPress Dashboard → Elementor → Templates**
2. Click **Import Template** (top right)
3. Click **Choose File** and select each JSON file
4. Click **Import Now**
5. Repeat for all 12 templates

**Result**: All templates will appear in **My Templates → Saved Templates**

---

### Step 2: Create Each Page

#### Create Homepage:
1. **Pages → Add New**
2. Title: "Home" (or your preferred homepage title)
3. Click **Edit with Elementor**
4. Click ➕ (Add Section)
5. Click **My Templates** tab
6. Insert templates in this order:
   - XCP_HomeP1_Hero_Gallery_FINAL
   - XCP_HomeP2_Parallax_Text_FINAL
   - XCP_HomeP3_Checkerboard_FINAL
   - XCP_HomeP4_Stats_Process_FINAL
   - XCP_HomeP5_Testimonials_CTA_FINAL
   - XCP_ContactForm_Section
7. Click **Publish**

#### Create About Page:
1. **Pages → Add New** → Title: "About"
2. **Edit with Elementor** → Add templates:
   - XCP_About_P1_Hero_Story
   - XCP_About_P2_Process_CTA
3. **Publish**

#### Create Services Page:
1. **Pages → Add New** → Title: "Services"
2. **Edit with Elementor** → Add template:
   - XCP_Services_P1_Packages
3. **Publish**

#### Create Portfolio Page:
1. **Pages → Add New** → Title: "Portfolio" (or "Gallery")
2. **Edit with Elementor** → Add templates:
   - XCP_Portfolio_P1_Hero_Filter
   - XCP_Portfolio_P2_Gallery_CTA
3. **Publish**

#### Create Contact Page:
1. **Pages → Add New** → Title: "Contact"
2. **Edit with Elementor** → Add template:
   - XCP_Contact_P1_Complete
3. **Publish**

---

### Step 3: Replace Placeholder Images

1. Click any image in Elementor editor
2. Click **Choose Image**
3. Upload your actual photos or select from Media Library
4. Adjust image settings if needed (size, alignment, etc.)
5. Repeat for all placeholder images

**Recommended Image Sizes:**
- **Hero backgrounds**: 1920×1080px (landscape)
- **Gallery boxes**: 800×600px (landscape)
- **Checkerboard images**: 1200×800px (landscape)
- **Portfolio images**: 1200×800px (mixed orientations)
- **Logo partners**: 300×200px (transparent PNG)

---

### Step 4: Set Homepage

1. Go to **Settings → Reading**
2. Select **A static page** under "Your homepage displays"
3. Choose your "Home" page from dropdown
4. Click **Save Changes**

---

### Step 5: Configure Form Submission

1. Go to homepage in Elementor editor
2. Click on the contact form
3. Click **Actions After Submit** in left panel
4. Add **Email** action
5. Enter your email address
6. Configure other settings as needed
7. **Update** page

## Troubleshooting

### Template Won't Import:
- Check Elementor version (should support v0.4 format)
- Ensure JSON file is not corrupted
- Try importing one at a time instead of batch
- Check server PHP memory limit (increase if needed)

### Images Not Showing:
- Verify image URLs are correct
- Upload images to WordPress Media Library
- Replace `[uc_local]` paths with actual URLs

### Form Not Submitting:
- Configure "Actions After Submit" in form settings
- Add Email action with valid recipient
- Test with dummy data
- Check spam folder for test emails

### Layout Issues:
- Clear Elementor cache: Elementor → Tools → Regenerate CSS
- Clear browser cache
- Check for plugin conflicts
- Verify theme compatibility with Elementor
